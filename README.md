# Montando / Creando las imagenes

1. Se debe ejecutar el siguiente comando _docker-compose up --build_

> Como alternativa, se puede dar clic derecho sobre el archivo _compose.yml_ y luego clic en "Compose Up"

Lo anterior crea los contenedores de cada servicio


## Acceder a la PHPMyAdmin
1. Luego de activar los contenedores, ir a [phpmyadmin](http://localhost:5050/)
* > Usuario: root
* > Contraseña: secret

## Acceder a la aplicación
1. En la ruta [localhost:5055](http://localhost:5055/) aparecerá la interfaz web
